Tic Tac Toe

How to Play:

1. The game is played on a 3 x 3 grid.

2. You are X, the other player is O. Players will take turns putting their marks in empty squares.

3. The first player to get 3 in a row (up, down, across, or diagonally) is the winner!

When all 9 squares are full, the game is over. If no player has 3 in a row, the game ends in a tie.

X goes first
